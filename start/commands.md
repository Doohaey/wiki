# 常用指令

在此会列举一些常用的指令，没有标明收费的即可免费使用。更多详细指令可以通过在游戏内输入 `/menu` 或 `/help` 的方式获取

* **`/spawn` 回到出生点**
* `/menu` 打开时钟菜单
* **`/bal` 查询银行余额**
* `/ptt` 查询在线时间
* **`/tpa [player]`、`/home` 等**[**传送命令**](../server-world/plugins/survice.md#chuan-song-xiang-guan)
* `/pay [玩家ID] [金额]` 可以支付给指定玩家相应数额的金钱
* `/t [玩家ID] [信息]` 可给指定玩家发送私信
